# Changelog

## 1.1.0

- Added Mythic+ and Raid URL modes.
- Added `/bwcl m`, `/bwcl m+`, and `/bwcl mplus` commands to switch to Mythic+ mode.
- Added `/bwcl raid` and `/bwcl r` commands to switch to Raid mode.
- Added `/bwcl status` command to show the current mode and zone.
- Updated Warcraft Logs URL generation to use different zone numbers depending on the selected mode.

## 1.0.0

- Initial release.
- Added WCL button to LFG applicant rows.
- Added copyable Warcraft Logs URL popup.
- Added `/bwcl Character-Realm` manual test command.
- Added `/wclcheck` manual refresh command.
